🧠 Health Risk Classification using Random Forest

This project implements a machine learning classification model to predict health risk using Random Forest with Randomized Search optimization.

📌 Project Overview

The model was developed as part of a master-level research project. It includes data preprocessing, feature selection, hyperparameter tuning, and performance evaluation.
